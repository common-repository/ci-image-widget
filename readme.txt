A simple image widget that allows you to display an image in any sidebar. The image can either link to another page or it can pop out in a lightbox.

== Description ==
Using this simple image widget you can display a clickable image in any sidebar.

You can set the image to link to a URL of your choice or make it pop-up in a lightbox.

The widget uses WordPress\' built in media manager to provide you with a familiar interface for image upload/selection.

You can select the widget\'s image size from the available drop-down menu which lists all available WordPress default image sizes.

== Installation ==
From within WordPress\' dashboard:

1. Go to Plugins -> Add New
2. Search for \"The Image Widget\"
3. Click \"Install\"
4. Click \"Activate\"
5. Go to *Appearance -> Widgets* or *Customize -> Widgets*
6. Drag the *The Image Widget* widget into the sidebar of your choice.
7. Set the title, the link URL and select an image from your media library.
8. Save and you are done.

Manually via FTP:

1. Upload the folder \'ci-image-widget/\' into the \'/wp-content/plugins/\' directory
2. Activate the plugin through the *Plugins* tab in WordPress
3. Go to *Appearance -> Widgets* or *Customize -> Widgets*
4. Drag the *The Image Widget* widget into the sidebar of your choice.
5. Set the title, the link URL and select an image from your media library.
6. Save and you are done.
